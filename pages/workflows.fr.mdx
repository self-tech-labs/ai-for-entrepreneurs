## Workflows Genilem

Dans ce chapitre nous mettons à disposition des exemples de workflows pour Genilem. Ces exemples sont des workflows de base qui peuvent être utilisés comme base pour s'aider dans la réalisation de tâches entrepreneuriales.

### Workflow 1 : rédaction article de blog

Ce workflow permet de rédiger un article de blog. Il est composé de 3 étapes :

1. Rédaction de l'article
2. Relecture de l'article
3. Publication de l'article


### Workflow 2 : création d'un site web

Ce workflow permet de créer un site web. Il est composé de 3 étapes :

1. Création du site web
2. Rédaction du contenu
3. Publication du site web
